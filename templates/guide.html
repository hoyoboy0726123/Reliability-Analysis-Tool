🔥 活化能 (Ea)
核心參數
定義：
失效機制啟動所需的最小能量障礙 (eV)
⚠️ 重要性：
Ea 是 Arrhenius 模型的核心，決定溫度加速效應的強度

元件/失效機制 Ea (eV) 說明
🔴 高溫極度敏感 (Ea ≥ 1.0)
IC 金-鋁鍵合線 1.0-1.26 金屬間化合物生長
塑膠材料老化 0.8-1.2 化學降解
金屬膜電阻 0.8-1.0 薄膜氧化漂移
🟡 半導體 IC 元件 (Ea = 0.6-1.0)
CPU/GPU 銅互連 ~0.9 電遷移 (EM)
MOS 氧化層 (TDDB) 0.6-0.9 內因性崩潰
鋁互連 0.55-0.6 舊技術 EM
🟢 業界默認值 (Ea = 0.7)
通用默認 0.7 JEDEC 標準，未知機制
碳成分電阻 0.7-0.9 阻值漂移
電容器 (AEC/TEC) 0.6-0.9 電解液蒸發
分立半導體 0.6-0.7 結點退化
🔵 機械/焊點 (Ea = 0.5-0.7)
焊點熱疲勞 0.5-0.7 溫度敏感度較低
焊點電遷移 0.6-0.8 高電流密度
⚫ 特殊機制
氧化層缺陷 (TDDB) 0.3 外因性，早期失效
熱載子注入 (HCI) -0.1 ~ -0.2 負Ea！低溫加速

Ea 代表什麼？

活化能是原子或分子系統從基態躍遷到能啟動特定失效過程所需的最小能量障礙。Ea 越高，該失效機制對溫度越敏感。

為什麼 Ea 如此重要？
  • 決定加速強度：Ea = 0.9 eV vs. 0.7 eV，溫度每升高 10°C，加速因子差異可達 30%
  • 不同機制有不同 Ea：一個元件可能有多個潛在失效模式，必須識別主導機制
  • 負 Ea 的挑戰：HCI 具負 Ea，低溫反而加速失效，顛覆常規認知
業界默認 0.7 eV 的局限性：

0.7 eV 是統計平均值，非物理精確值。對現代高性能 IC (CPU/GPU)，錯用 0.7 eV 會嚴重低估溫度影響。

⚠️ 關鍵原則：

準確的可靠度評估必須基於特定失效機制的 Ea，而非通用默認值。當無實驗數據時，參考同類元件文獻或材料供應商數據。

方法 1：Arrhenius 繪圖 (實驗法)

1. 選擇至少 3 個溫度 (如 85°C, 100°C, 125°C)
2. 每個溫度測試 20-30 樣品，記錄失效時間
3. 繪製 ln(MTTF) vs. 1/T (Kelvin)
4. 直線斜率 = -Ea/k，求出 Ea

方法 2：查詢文獻與標準

• JEDEC Publication 122C：提供多種失效機制 Ea 範圍
• 材料供應商數據表
• 同類產品可靠度報告
• 學術論文 (IEEE, IEDM)

方法 3：失效物理分析 (PoF)

通過失效分析 (FA) 確定主導失效機制：
• SEM/TEM：觀察微觀結構
• 電性分析：確定失效模式
• 匹配已知機制的 Ea

⚠️ 注意：

• 零失效測試：JEDEC 建議使用 Ea = 1.0 eV (保守)
• 多重機制：總失效率是所有模式之和
• 溫度範圍：Ea 在不同溫度範圍可能變化

🌡️ 溫度加速因子
Arrhenius
公式：
AFT = exp[(Ea/k) × (1/Tuse - 1/Talt)]

參數 說明 典型值
Tuse 使用溫度 (°C) 25-35°C
Talt 測試溫度 (°C) 60-125°C
Ea 活化能 (eV) 0.3-1.2 eV

  • • 電子元件：Ea = 0.7-1.0 eV
  • • 焊點：Ea = 0.5-0.8 eV
  • • 塑膠材料：Ea = 0.8-1.2 eV
  • • JEDEC 測試溫度：85°C, 125°C, 150°C

步驟 1：設計實驗

選擇至少 3 個溫度水平（例如：85°C, 100°C, 125°C），每個溫度測試 20-30 個樣品

步驟 2：收集失效數據

記錄每個樣品的失效時間 (TTF)

步驟 3：Arrhenius 繪圖

繪製 ln(MTTF) vs. 1/T，斜率 = -Ea/k

⚠️ 注意：

如果無實驗數據，可使用材料供應商提供的 Ea 值，或參考同類產品文獻

  • → NASA SSRI: Accelerated Life Testing
  • → JEDEC Standards (JESD22-A103)
  • → Arrhenius Equation (Wikipedia)
💧 濕度加速因子
Peck Model
公式：
AFRH = (RHalt / RHuse)n

參數 說明 典型值
RHuse 使用濕度 (%RH) 50-70%
RHalt 測試濕度 (%RH) 85-95%
n 濕度指數 2.0-3.0

  • • 一般電子產品：n = 2.0-2.5
  • • PCB 腐蝕：n = 2.5-3.0
  • • JEDEC HAST：130°C / 85% RH
  • • THB (溫濕偏壓)：85°C / 85% RH

步驟 1：固定溫度

選擇一個測試溫度（例如 85°C）

步驟 2：濕度變化

測試 3-4 個濕度水平（例如：60%, 75%, 85%, 95%）

步驟 3：參數估計

繪製 ln(MTTF) vs. ln(RH)，斜率 = -n

⚠️ 注意：

濕度與溫度常有交互作用，建議使用 Peck 完整模型（溫度+濕度）

  • → JESD22-A110 (HAST)
  • → JESD22-A101 (THB Test)
⚡ 電壓加速因子
IPL Model
公式：
AFV = (Valt / Vuse)β

參數 說明 典型值
Vuse 額定電壓 (V) 產品規格
Valt 測試電壓 (V) 1.1-1.5× Vrated
β 電壓指數 1.0-5.0

  • • 保守估計：β = 1.0
  • • 電介質擊穿：β = 2.4-2.7
  • • 電遷移：β = 1.5-3.0
  • • 測試電壓：不超過 1.5× Vrated

步驟 1：選擇電壓水平

通常選擇 3-4 個電壓水平（例如：1.0V, 1.1V, 1.2V, 1.3V）

步驟 2：固定溫度

在相同溫度下進行測試

步驟 3：IPL 繪圖

繪製 ln(MTTF) vs. ln(V)，斜率 = -β

⚠️ 注意：

過高電壓可能改變失效機制，建議不超過額定電壓的 150%

  • → Analog Devices: Voltage Acceleration
  • → Inverse Power Law (ReliaSoft)
🔄 熱循環加速因子
Coffin-Manson
公式：
AFTC = (ΔTalt/ΔTuse)β × (falt/fuse)α

參數 說明 典型值
ΔTuse 使用溫度範圍 (°C) 50-80°C
ΔTalt 測試溫度範圍 (°C) 100-165°C
fuse 使用頻率 (cycles/hr) 0.042 (1次/天)
falt 測試頻率 (cycles/hr) 1-3 cycles/hr
α 頻率指數 0.33
β 溫度指數 1.9 (焊點)

  • • 錫鉛焊點 (SnPb)：β = 1.9, α = 0.33
  • • 無鉛焊點 (SAC)：β = 2.65, α = 0.33
  • • 陶瓷材料：β = 5.0-6.0
  • • JEDEC條件：-55°C to +125°C (ΔT=180°C)
  • • IPC-9701：板級可靠度標準

步驟 1：定義循環剖面

確定最高/最低溫度和駐留時間（例如：-40°C/30min → +125°C/30min）

步驟 2：測試多個ΔT

使用 3 種不同的 ΔT（例如：100°C, 140°C, 180°C）

步驟 3：記錄失效循環數

定期檢查（例如：每100循環）直到失效

步驟 4：參數擬合

使用 Coffin-Manson 模型擬合 β 和 α

⚠️ 注意：

頻率效應：過快循環可能低估現場壽命（蠕變效應）

  • → JESD22-A104C (Thermal Cycling)
  • → IPC-9701 (Board-Level Reliability)
  • → Indium Corporation: TC Acceleration
📳 振動加速因子
IPL Model
公式：
AFVIB = (Galt / Guse)n

參數 說明 典型值
Guse 使用加速度 (g) 0.5-2.0 g
Galt 測試加速度 (g) 10-20 g
n 功率指數 6.0-12.0

  • • 焊點疲勞：n = 8.0-10.0
  • • PCB 裂紋：n = 6.0-8.0
  • • 一般桌上環境：Guse = 0.5-1.0 g
  • • 汽車環境：Guse = 2.0-5.0 g
  • • JEDEC測試：10-20 g (200-2000 Hz)

步驟 1：振動頻譜分析

使用加速度計測量實際使用環境的振動頻譜

步驟 2：設計振動測試

選擇 3-4 個加速度水平（例如：5g, 10g, 15g, 20g）

步驟 3：振動台測試

掃頻或隨機振動測試，記錄失效時間

步驟 4：參數估計

繪製 ln(MTTF) vs. ln(G)，斜率 = -n

⚠️ 注意：

振動測試需要專業設備（振動台），建議委託測試實驗室

  • → JESD22-B103B (Vibration Testing)
  • → MIL-STD-810 (Military Standard)
☀️ UV輻射加速因子
實驗比對
公式：
AFUV = tfield / taccelerated

參數 說明 典型值
tfield 現場暴露時間 (小時) 8760 (1年)
taccelerated 達到相同退化的測試時間 500-2000小時

  • • QUV測試：1000小時 ≈ 戶外 1-2年
  • • Xenon Arc：1000小時 ≈ 戶外 6-12個月
  • • 塗料/塑膠：AF = 5-15 (產品特定)
  • • 測試條件：340nm, 0.89 W/m², 60°C

步驟 1：定義失效標準

例如：光澤度降低50%、顏色變化ΔE≥5、拉伸強度下降20%

步驟 2：現場暴露

將樣品放置在實際戶外環境（如屋頂），定期檢測性能

步驟 3：加速測試

使用 QUV 或 Xenon 測試箱，控制：
• 光譜輸出（模擬太陽光）
• 溫度（60-70°C）
• 濕度（50-75% RH）
• 循環模式（UV/冷凝）

步驟 4：比對計算

當兩者達到相同失效標準時：
AF = tfield / taccelerated

⚠️ 重要限制：

• UV加速因子無法純理論計算，必須實驗確定
• 產品特定性：不同材料的AF差異很大
• 高度加速可能改變失效機制
• 建議查詢材料供應商數據或同類產品文獻

  • → ASTM G155 (Xenon Arc Exposure)
  • → ISO 4892 (Plastics Weathering)
  • → Atlas: Weathering Basics
  • → Accelerated Weathering Principles
🧪 化學濃度加速因子
IPL Model
公式：
AFCHEM = (Calt / Cuse)n

參數 說明 典型值
Cuse 使用濃度 (相對單位) 1.0 (基準)
Calt 測試濃度 (相對單位) 3-10倍
n 功率指數 1.5-3.0

  • • 金屬腐蝕：n = 2.0-2.5
  • • 聚合物降解：n = 1.5-2.0
  • • 鹽霧測試：5% NaCl溶液
  • • 相對濃度：以標準環境為基準 (1.0)

步驟 1：定義基準濃度

確定實際使用環境的化學物質濃度作為 Cuse = 1.0

步驟 2：濃度梯度測試

測試 3-4 個濃度水平（例如：1×, 3×, 5×, 10×）

步驟 3：固定溫度

在相同溫度下進行測試（通常結合 Arrhenius 模型）

步驟 4：參數估計

繪製 ln(MTTF) vs. ln(C)，斜率 = -n

步驟 5：濃度監測

使用 Beer-Lambert 定律監測溶液濃度：
A = ε × l × c

⚠️ 注意：

• 化學加速常與溫度有交互作用
• 過高濃度可能改變失效機制
• 需控制溶液 pH 值和氧氣濃度

  • → ASTM B117 (Salt Spray Test)
  • → Inverse Power Law (ReliaSoft)
☢️ 輻射劑量加速因子
TID Model
公式：
AFRAD = (Dalt / Duse)n
⚠️ 特殊應用：
僅適用於太空任務或核環境應用

參數 說明 典型值
Duse 使用環境累積劑量 10-150 krad
Dalt 測試累積劑量 100-300 krad
劑量率 測試劑量率 50-200 krad/hr
n 劑量敏感度指數 0.5-2.0
劑量敏感度指數 n 的意義：
  • n < 1：次線性，劑量率效應顯著（退火恢復）
  • n = 1：線性，標準 TID 模型
  • n > 1：超線性，高劑量協同效應

太空環境累積劑量 (TID)：
  • • LEO 低軌：5-15 krad/year
  • • GEO 同步軌：30-100 krad/year
  • • 極地軌道：10-20 krad/year
  • • 太陽風暴：可達數倍正常值
典型任務累積劑量：
  • • 5年 LEO 任務：50-75 krad
  • • 15年 GEO 任務：450-1500 krad
  • • 核設施儀表：視輻射源距離而定

步驟 1：確定任務劑量

根據軌道高度、任務時長、屏蔽厚度估算累積 TID
工具：SPENVIS (ESA), CREME96 (NASA)

步驟 2：選擇輻射源

• Co-60 γ 射線：能量 1.17-1.33 MeV
• 電子束：可調能量，適合 ELDRS 研究
• X 射線：低能量，模擬太陽耀斑

步驟 3：劑量率測試

至少測試 3 個劑量率（例如：10, 50, 300 rad/s）
確定是否存在 ELDRS (Enhanced Low Dose Rate Sensitivity)

步驟 4：劑量梯度測試

測試多個累積劑量點（例如：10, 30, 50, 100 krad）
繪製失效參數 vs. 累積劑量，擬合 n 值

步驟 5：退火效應

輻照後進行室溫或高溫退火（168小時）
評估參數恢復程度，確定長期效應

⚠️ 重要限制：

• 劑量率效應：低劑量率可能更危險（ELDRS）
• 器件相關性：不同製程差異極大
• 偏壓影響：必須在工作偏壓下測試
• 溫度交互：低溫可能抑制退火恢復

  • → MIL-STD-883 Method 1019 (TID Testing)
  • → ESCC Basic Specification No. 22900
  • → SPENVIS (Space Environment Info System)
  • → CREME96 (Cosmic Ray Effects)
⚙️ Eyring 模型 (進階)
應力交互
廣義 Eyring 公式：
t = A × (1/S)B × eEa/kT × eD×S/T
🎓 進階用途：
用於處理應力之間有交互作用的複雜情況

使用情境：
  • 溫度+電壓交互：高溫加劇電遷移效應
  • 溫度+濕度交互：溫度越高，濕度影響越顯著
  • HAST測試：高加速應力測試（T-RH-V同時作用）
何時不需要：
  • 應力相互獨立（大多數情況）
  • 單一應力測試
  • 消費性產品（簡化模型已足夠）

變數 說明
t 產品壽命
T 絕對溫度 (Kelvin)
S 非熱應力（電壓、濕度等）
Ea 活化能 (eV)
A, B, D 模型常數（需實驗確定）
D 交互作用項常數（關鍵）

步驟 1：階乘實驗設計 (DOE)

設計多水平多因子測試矩陣：
例如：3個溫度 × 3個電壓 = 9組實驗

步驟 2：每組測試

每個條件至少 20-30 個樣品

步驟 3：收集失效數據

記錄所有失效時間 (TTF)

步驟 4：參數擬合

使用專業軟件（ReliaSoft, Minitab）進行最大似然估計 (MLE)

步驟 5：模型驗證

在新的應力水平下測試，驗證預測準確性

⚠️ 複雜度警告：

• 需要大量實驗數據（通常 >200 樣品）
• 需要專業統計軟件
• 適用於高可靠度產品（軍用、航太）
• 大多數消費性產品使用簡化模型即可
